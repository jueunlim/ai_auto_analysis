# AI 자동화 데이터 분석 실습

이 레포지토리는 [패스트캠퍼스 AI 자동화 강의](https://fastcampus.co.kr/data_online_cursormcp)를 학습하며 데이터 분석 실습을 실제로 수행하고 정리하는 개인 프로젝트 레포지토리입니다.

강의 중 수행한 각 프로젝트는 디렉토리로 구성되어 있으며, 각 디렉토리에는 해당 실습의 코드, 결과, 시각화 자료가 포함되어 있습니다.

## 개발환경
- 언어 : Python
- IDE : Cursor 

## 프로젝트 목차

### [01_bank_da](./01_bank_da/)
은행 마케팅 데이터셋을 활용한 탐색적 데이터 분석(EDA) 및 머신러닝 모델링 프로젝트입니다. 고객의 정기예금 가입 여부를 예측하는 분류 모델을 구축하고, Random Forest와 Gradient Boosting 모델을 비교 분석했습니다.

- **데이터**: 41,188행 × 21열의 은행 마케팅 데이터
- **목표**: 고객의 정기예금 가입 여부 예측 (이진 분류)
- **주요 작업**: EDA, 특성 분석, 모델링 및 성능 평가
- **최종 성능**: Gradient Boosting (AUC-ROC: 0.9536)
